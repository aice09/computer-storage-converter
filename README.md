# Computer Data Storage Converter & Memory Mapper

This interactive tool allows you to convert between all major computer data storage units, including bit, nibble, byte (B), binary units (KiB, MiB, GiB…) and decimal units (KB, MB, GB…).

Features include:
- Real-time conversion between binary (IEC) and decimal (SI) units
- Memory mapping table showing KiB ↔ KB, MiB ↔ MB, etc.
- Hierarchy diagram of data units with animated flow using D3.js
- Historical background of bits, bytes, IEC & SI units
- References to ISO/IEC and Wikipedia standards
- Tooltips and explanations for easy understanding of base, binary, and decimal units
-Perfect for students, developers, and IT professionals who want to understand the relationships between digital storage units and memory sizes.
